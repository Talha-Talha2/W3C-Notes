Page Title

A page title is crucial to a website it showcases the page content whilst distinguishing it from the other pages. The page title really helps the user understand what web page he is on and is often the same as the main heading. For example, for a specific recipe such as Noodles its page title will be the same as the Noodles (the heading

Headings

Headings are used to group related paragraphs and clearly help the user understand the content, while describing its sections. An effective heading gives a good summary to the content

Link Text

Link text has to be effective at describing the contents of the link target. Avoid using link text, such as “click here” without indicating relevant information. Add information such as Click here to view the documents so the user understands where the link takes you.

Meaningful

For every image, write text which provides information or the function of the image, this helps the user better interpret the function

Create Transcripts and captions for multimedia

Providing a transcript further helps the user better interpret specific audio files and enables them to get more out of them.

Provide clear insturctions

Provide clear instructions about each aspect of functionality for your page helping the user understand this can be achieved by avoiding unnecessary technical language. And showcasing specific input requirements, such as date formats.

Keep content clear concise

Use simple and straightforward language.
